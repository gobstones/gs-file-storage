An example of
<gs-file-storage>:

File Explorer

  • Create files and see them into left file-explorer.

  • Also refresh page and see last file content steel there.

  • Create a variable and initialize gs-file-system like var this.storage = GsFileStorage() to use it.

  • gs file system is located in window.GS.GSFILESTORAGE and works like a singleton

  • gs-file-system let you save gs-files when create a gs-file, create a new GSFile("name") and then gs-file.saveIt() an this save it into localStorage On save trigger event change in file and if file is new gs-file-system trigger listchange event.
